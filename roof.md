# Shumoukh Roof - نظام إدارة مشاريع الأسقف

## نظرة عامة
تطبيق ERP لإدارة مشاريع الأسقف والمباني، يوفر حسابات دقيقة وأدوات ثلاثية الأبعاد.

## الميزات الرئيسية
- حساب تكلفة الأسقف والمباني
- عرض ثلاثي الأبعاد للأسقف (BabylonJS)
- إدارة المشاريع والعمال والموردين
- إصدار الفواتير والتقارير
- نظام إشعارات
- دعم متعدد اللغات (عربي/إنجليزي)

## التقنيات
- React 19 + TypeScript
- Vite 8
- Capacitor (تطبيق جوال)
- Firebase (المصادقة وقاعدة البيانات)
- BabylonJS (الرسومات ثلاثية الأبعاد)
- Zustand (إدارة الحالة)
- Tailwind CSS
